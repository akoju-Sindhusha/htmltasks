VISUAL STUDIO(VS CODE):-

VS Code supports almost every major programming language. Several ship in the box, like JavaScript, TypeScript, CSS, and HTML, but extensions for others can be found in the VS Code Marketplace.

Visual Studio Code combines the simplicity of a source code editor with powerful developer tooling, like IntelliSense code completion and debugging. First and foremost, it is an editor that gets out of your way. The delightfully frictionless edit-build-debug cycle means less time fiddling with your environment, and more time executing on your ideas. Available for macOS, Linux, and Windows.

FEATURES OF VS :-

    1.Debugging: Step through source code, inspect variables, view call stacks, and execute commands in the console
    2.Syntax highlighting: Highlights syntax for different programming languages
    3.Code completion: Provides intelligent suggestions for code completion
    4.Code snippets: Insert blocks of code
    5.Code refactoring: Refactor code
    6.Outlining: Expand and collapse blocks of code
    7.IntelliSense: Learn more about code
    8.Object Browser: Learn more about code
    9.Call Hierarchy: Learn more about code
    10.Go To: Find code
    11.Go To Definition: Find code
    12.Find All References: Find code
    13.Generate From Usage: Generate code
    14.Integrated terminal: Execute commands and run scripts without leaving the editor
    15.Git integration: Manage source code changes and collaborate with other developers

SHORTCUTS FOR VS CODE:-

S.NO SHORTCUTS DESCRIPTION
1. ctrl+S save the current file
2. ctrl+C copy the selected text or item
3. ctrl+V paste the coppied content
4. ctrl+X cut the selected text or item
5. ctrl+Y undo the last action
6. ctrl+Z redo the previously undone action
7. ctrl+A select all the content in the current file
8. ctrl+Plus(+) zoom in the editor view
9. ctrl+minus(-) zoom out the editor view
10. ctrl+shift+alt+down add cursor to multiple lines for simultaneous editing
11. alt+up arrow move the selected line or block of code one line up
12. alt+down arrow move the selected line or block of code one link up
13. ctrl+shift+P open command pallete
14. ctrl+shift+N new window
15. ctrl+shift+w close window
